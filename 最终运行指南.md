## 🚀 如何运行

### 步骤 1: 打开 Windows PowerShell
- 按 `Win + X`，选择 "Windows PowerShell" 或 "终端"

### 步骤 2: 切换到桌面
```powershell
cd C:\Users\Administrator\Desktop
```

### 步骤 3: 安装依赖（首次运行）
```powershell
pip install selenium openpyxl
```

### 步骤 4: 运行脚本
```powershell
python deepseek_listing_generator_v5.py
```

### 步骤 5: 选择处理范围
- 输入 `2` - 测试前 3 个（**强烈推荐先测试**）
- 输入 `1` - 处理全部
- 输入 `3` - 自定义范围

### 步骤 6: 确认 DeepSeek
浏览器会自动打开 DeepSeek：
- 确保已登录
- 确保 Prompt 已配置好
- 按回车继续
